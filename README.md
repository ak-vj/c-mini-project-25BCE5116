# Hospital Patient Registration & Follow-up Tracker

Menu-driven C program to manage patient records in a small clinic.

## Features
- Register patient
- Add visit record
- Search patient by ID
- Show visit history
- Detect frequent visitors
- File handling for saving/loading data

## Compile and Run

gcc main.c
./a.out

## Sample Menu

1 Register Patient
2 Add Visit
3 Search Patient
4 Show Visit History
5 Show Frequent Visitors
6 Save and Exit
